# 3Sum

## 题目描述

给你一个整数数组 `nums`，判断是否存在三元组 `[nums[i], nums[j], nums[k]]` 满足 `i != j、i != k 且 j != k`，同时还满足 `nums[i] + nums[j] + nums[k] == 0`。

请你返回所有和为 `0` 且不重复的三元组。

**注意：**答案中不可以包含重复的三元组。

## 思路分析

### 方法一：排序 + 双指针

1. 先排序数组
2. 固定第一个数，用双指针寻找另外两个数
3. 注意跳过重复元素

- 时间复杂度：$O(n^2)$
- 空间复杂度：$O(1)$
